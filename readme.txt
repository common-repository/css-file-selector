Select files CSS and/or write CSS rules to any single page or post

== Description ==
A simple Wordpress plugin that allows users to select files CSS and/or write CSS rules to any single page or post

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload entire `css-file-selector` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it, you'll see a `Select CSS files and/or write your custom CSS rules` metabox on all add or edit posts/pages.
